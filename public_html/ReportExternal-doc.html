Internal/External Network Traffic - Description

This graph shows IP traffic seen flowing by your network connection. Local IP addresses (as determined by the LOCALRANGE variable in the ipaudit-web.conf file) are considered internal (local). All others are considered external (remote).

The line marked External shows the bandwidth used by packets with both external source and destination addresses. Such data might be caused by multicast traffic. If a local host is sending data with a spoofed external source address it may show up as external traffic.

The line marked Internal shows packet throughput with both ip addresses internal to your network.

The traffic is sampled over 30 minute intervals.
